## Objetivo del Programa

Se desarrolló un software en Python que simula el juego clásico de Piedra, Papel o Tijera, aplicando los conceptos fundamentales de lógica de programación vistos durante el módulo.


## ⚙ Principales Funcionalidades

- Le da opción al usuario de seleccionar una opción: piedra, papel o tijera.
- Luego la computadora genera una opción aleatoria utilizando el módulo `random`.
- Posteriormente se comparan ambas elecciones para determinar el ganador.
- Se utilizó una estructura condicional para definir el resultado.
- También implementé un marcador acumulativo de puntos.
- Un apartado donde se guarda un historial de las partidas jugadas.
- El programa permite repetir el juego hasta que el usuario decida finalizar.


## Conceptos Aplicados
